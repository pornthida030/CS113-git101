การสร้าง/แยกbranch
1. git clone URL เป็นการนำโฟเดอร์เก่าในgithubมาใส่ (เปลี่ยนdirectory)
2. git branch ชื่อbranchที่ต้องการใส่ เพื่อทำการสร้าง branch 
เพิ่มเติม
-git branch เป็นการตรวจสอบว่ามีbranchใดบ้าง ถ้ามี*แสดงว่าเรากำลังอยู่ในbranchนั้น
-git checkout ชื่อbranchที่ต้องการเข้า

--------------------------------------------------------------------------------------------------------------------------------------------------------------
การMerge branch not conflict (ทำการสร้างbranchไปแล้วถึงทำได้)
1.git checkout ชื่อbranchที่ต้องการเข้าทำการเพิ่มและcommitแต่ละไฟล์
2.git push origin ชื่อbranchที่ต้องการเข้า(ต้องมีการcheckoutในbranchที่ต้องการก่อนถึงทำได้)
3.ต้องเป็นbranchปลายทางก่อน(git checkout ชื่อbranch)แล้วก็รวมให้อยู่ในbranchเดียวกันโดย git merge branchที่จะรวม
เสร็จแล้วก็pushไปที่github

--------------------------------------------------------------------------------------------------------------------------------------------------------------
การPull ไม่มีconflict
1.ทำการcloneไว้อีกเครื่องและ push
2.เพิ่มโฟเดอร์ใหม่
3.เครื่องเก่าทำการcd โฟเดอร์เก่าและใช้คำสั่งgit pull origin ชื่อbranch เพื่อดึงการเปลี่ยนแปลง

--------------------------------------------------------------------------------------------------------------------------------------------------------------
การMerge conflict(ความขัดแย้ง)
1.สร้างbranch สลับbranchเลย ใช้ git checkout -b ชื่อbranch แล้วก็สร้างไฟล์และadd and commit และแก้ไขไฟล์ในอีกbranchหนึ่ง แล้วทำการ add and commit 
2.ทำการMerge ซึ่งจะไม่สำเร็จต้องทำการเปิดไฟล์นั้นแล้วเลือกข้อความที่เราต้องการ หลังจากแก้ไขเสร็จก็ add and commit and push

--------------------------------------------------------------------------------------------------------------------------------------------------------------
การPull conflict
1.ทำในเครื่องใหม่cd โฟเดอร์ใหม่ -> git pull origin ชื่อbranch 
2.แก้ไขไฟล์ที่สร้างขึ้นของโฟเดอร์เก่า add and commit and push
3.เครื่องเก่าcd โฟเดอร์เก่า add and commit ทำการpull แก้ไขไฟล์ในโฟเดอร์เก่า add and commit and push 
4.เครื่องใหม่ cd โฟเดอร์ใหม่ ทำการ pull
